1. FLASH存储器

SRAM和DRAM都是随机读写存储器，他们的特点就是可读可写。而ROM则是只读存储器,它又分为掩模ROM和可编程ROM,其中可编程ROM学习了单片机ARM就会非常熟悉多次编程的EPROM以及E2PROM.(PROM就是一次性编程)
FLASH存储器是在EPROM存储元基础上发展起来的,由单个MOS晶体管,除漏极D和源极S,还有一个控制栅和浮空栅。

  • 编程操作：也就是写操作，因为擦除操作的时候控制栅不加正电压，所以存储元的原始状态均是’1’状态，而写操作就是使得存储元状态变为’0’即在控制栅上加正电压，为存储元的浮空栅补充电子。
  • 读取操作：写操作完成那么控制栅上就加上了正电压。而浮空栅上的负电荷量将决定是否可以开启MOS管。当MOS导通，电路检测到电流表示存储元存1,若电路检测无电流则为0，通过写操作我们知道存储元由1变为0之后，浮空栅带负电，那么控制栅上的正电压不足以克服浮动删上的负电量，所以晶体管不能导通。
  • 擦除操作：擦除操作时所有存储元的浮空栅中的负电荷都要放出去，为此晶体管的源极S上加上正电压，吸收浮空栅中的电子。

4. 事件模块(Events模块)

EventEmit 类下生成事件对象， life = new EventEmit();
on方法监听事件，emit方法触发事件；
life.setMaxListeners(11);设置事件最大监听（针对每种事件）默认为10个
官方说明超过十个会造成内存泄露，最好不要超过十个
life.on(‘eventname’,function(arg){});监听事件和事件触发后执行的函数
life.emit( ‘eventname’, arg)； // 触发的事件和需要传递的形参
emit 方法的返回值为true 或 false, 如果事件监听过为true,没哟监听过为false
