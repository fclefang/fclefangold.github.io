  • 每一个应用程序中都有共性的功能，可以将这些功能进行抽取独立封装以便复用。

虽然可以通过建立ArrayTool的对象使用这些工具方法从而对数组进行操作。但是对象是用于封装数据的，而ArrayTool并没有封装特有的数据，当我们操纵数组中的每一个方法都没有用到ArrayTool的特有数据。这时候就考虑道让程序更加严谨，是不需要对象的：可以通过将类中的方法都用static修饰，这样可以直接通过类名调用即可。
将方法都静态后，可以方便使用，但是该类还是可以被建立对象的，所以为了更加严谨(强制让该类不能建立对象)可以通过将构造函数私有化完成(当我们new一个对象时就会找对应的空参数构造函数初始化，所以一旦将其私有化之后将不能建立对象)。

  • 如何将工具类变成说明文档:dos命令：javadoc -d -author -version ArrayTool.java该类必须public，否则生成说明文档报错！
  • 静态代码块：类加载的时候就会调用static{}

要知道只有用到类里面的内容时才会涉及到类的加载，比如：new StaticCode();StaticCode.show();而如果只是引用类的话并不加载也就不会调用static{}了。所以类有没有加载可以通过静态代码块来验证。

  • 对象的初始化过程：Person p = new Person("ZhangShan,20");
    1.因为new用到了Person.calss.所以会先找到.class文件并加载到内存中。
    2.执行该类中的static代码块，如果有的话给类进行出赤化。
    3.在堆内存中开辟空间，分配内存地址。
    4.在堆内存中建立对象的特有属性并进行默认初始化(对象初始化显示)。
    5.对对象进行构造代码块初始化。
    6.对对象进行对应的构造函数初始化。
    6.将内存地址赋给栈内存中的p变量。
