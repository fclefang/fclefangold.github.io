安装numpy模块

(1)numpy模块资源文件下载完成（必须符合本机系统）.
(2)dos界面中cd进入python安装文件下的Script文件中，然后执行pip install 模块资源目录\模块全名，小技巧：在Script目录下执行dos命令pip list可以查看pip安装的模块名称。

交互环境下import模块包

  • 当我们想import一个工具模块时，出现错误的情况要么是模块不存在，要么就是默认的系统路径中找不到这个模块，所以可以通过import sys这个sys包来进行系统路径的配置。print sys.path可以查看到环境默认的一些路径变量，也就是说如果将工具模块包放置在这些路径下可以直接通过import 模块名。或者sys.path.append('模块路径')来添加你的模块所在的路径，这样系统在编译时就会在你添加的路径下进行模块的查找并导入。
  • 在目录中建一个文件名为init.py的文件，就可以直接按照路径名import模块。有了这个文件，我们就可以导入这个目录下的文件了

实施KNN算法

利用kNN算法为每组数据分类，对未知属性的数据集中的每个点依次执行一下操作：
(1)计算已知类别数据集中的点与当前点之间的距离。
(2)按照距离递增次序排序。
(3)选取与当前点距离最小的k个点。
(4)确定前k个点所在类别的出现频率。
(5)返回前k个点出现频率最高的类别作为当前点的预测分类。

kNN算法python代码解析：

  • dataSetSize = dataSet.shape[0]:shape返回矩阵的[行数，列数]，那么shape[0]获取数据集的行数，行数就是样本的数量。
  • tile(inX,(dataSetSize,1)):tile属于numpy模块下边的函数,tile（A, reps）返回一个shape=reps的矩阵，矩阵的每个元素是A。这个地方就是为了把输入的测试样本扩展为和dataset的shape一样，然后就可以直接做矩阵减法了。
  • sqDiffMat = diffMat**2：diffMat是一个矩阵，矩阵2表示对矩阵中的每个元素进行2操作，即平方运算。
  • sqDistances = sqDiffMat.sum(axis=1)：axis=1表示按照横轴，sum表示累加，即按照行进行累加。
  • distance = sqDistance ** 0.5：对平方和进行开根号。到这一步为止算法实现欧式距离计算。要记住的是转换为矩阵格式是为了方便计算。
  • distances.argsort()：按照升序进行快速排序，返回的是原数组的下标。 比如，x = [30, 10, 20, 40] ，升序排序后应该是[10,20,30,40],他们的原下标是[1,2,0,3]，那么，numpy.argsort(x) = [1, 2, 0, 3] 。
  • distances.argsort():按照升序进行快速排序，返回的是原数组的下标。classCount={}存放最终的分类结果及相应的结果投票数。再利用for循环得到key各自的票数value值，最后对分类结果进行排序获得票数最多的分类key.
