2.3.3 ROC和AUC

对于二分类问题，可将样例根据其真实类别与学习器预测类别的组合划分为真正类(true positive)、假正例(false postive)、真反例(true negative)、假反例(false negative)四种情形。很多学习器是为测试样本产生一个实值或概率预测，然后将这个预测值与一个分类阀值(threshold)进行比较，若大于阀值则为正类，否则为反类。根据概率预测结果将测试样本进行排序，这样分类过程就是在这个序列中找到一个截断点，前一部分作为正例，后一部分为反例(不同的应用任务采用不同的截断点)。排序质量的好坏体现了综合考虑学习器在不同任务下“期望泛化性能”的好坏，ROC(Receiver Operating Characteristic受试者工作特征)曲线就是从该角度来研究学习器泛化性能的有力工具。ROC和PR使用查准率和查全率不同，ROC使用TPR和FPR(见书33页)

2.4.2 交叉验证t检验

对两个学习器A和B，若我们使用k折交叉验证法得到测试错误率，可用k折交叉验证“成对t检验”来进行比较检验，这里的基本思想是若两个学习器的性能相同，则它们使用相同的训练/测试集得到的测试错误率应相同。具体说对k折交叉验证产生的k对测试错误率，先对没对结果求差，若性能相同，则差值均值应该为0.因此根据差值来对学习器A与B性能相同这个假设做t检验，计算出均值和方差。想要进行有效的假设检验，一个重要前提就是测试错误率均为泛化错误率的独立采样。然而因为样本有限，在用交叉验证等实验估计时，不同轮次的训练集会有一定程度的重叠，这就使得测试错误率实际上并不独立，会导致高估计假设成立的概率。

第二章课后习题

  1. 数据集包含1000个样本，其中500个正例和500个反例，将其划分为包含70%样本的训练集和30%样本的测试集用于留出法评估，试估算共有多少种划分方式。
    答：根据分层采样可得出训练集：350正例+350反例；测试集：150正例+150反例.所以划分方式为：$C{500}^{350}$*$C{500}^{350}$.
  2. 数据集包含100个样本，其中正反例各一半，假定学习算法所产生的模型是将新样本预测为训练样本数较多的类别(训练样本数相同时进行随机猜测)，试给出用10折交叉验证法和留一法分别对错误率进行评估所得的结果。
    答：(1):10折交叉验证法：由于正反例数目相同进行随机预测所以错误率为50%。
    (2):留一法：因为模型预测为训练样本数多的类别，所以留一法后留下的测试例必定是和预测类别相反的，所以错误率为100%。
  3. 若学习器A的F1值比学习器B高，试析A的BEP值是否也比B高？
    答：首先P-R图就是查准率-查全率曲线，BEP(平衡点Break-Even Point)是查准率=查全率时的取值。F1度量=(2乘P乘R)/(P+R)=(2乘TP)/(样例总数+TP-TN).因为F1$_A$>F1$_B$即(2乘P$_A$乘R$_A$)/(P$_A$+R$_A$)>(2乘P$_B$乘R$_B$)/(P$_B$+R$_B$).当P$_A$=R$_A$，P$_B$=R$_B$时，P$_A$>P$_B$,所以A的BEP值更高。
  4. 试述真正例率(TPR),假正例率(FPR)与查准率P，查全率R之间的联系。
    答：真正例率是所有真实正例中真正例的比例；
    假正例率是所有真实反例中假正例的比例；
    查准率是所有预测正例中真正例的比例；
    查全率是所有真实正例中真正例的比例；
    查全率与真正例率相等即R=TPR。
  5. 试证明AUC = 1-l$_{rank}$.
    答：与P-R图类似，若一个学习器的ROC曲线被另一个学习器的曲线完全包住则后者性能优于前者，若两个曲线发生交叉则可通过比较ROC曲线下的面积即AUC.
  6. 试述错误率与ROC曲线的联系.
    答：参考真假正例率的定义，由于样本中正例和反例的比例是确定的。因此以某个样本的预测值为阈值时（即对应ROC曲线中的一个点），该点的真正例率越高，假正例率越低，则错误率越低，因此越靠近(0,1)点错误率越低。
  7. 证明任意一条ROC曲线都有一条代价曲线与之相对应，反之亦然。
    答：ROC曲线上的坐标为(TPR,FPR),则可相应计算出假反例率FNR=1-TPR，然后在代价平面上绘制一条从(0，FPR)到(1，FNR)的线段。
  8. 试述X$^2$卡方检验过程.
    答：根据概率论与数理统计中的内容（交大版本，P239）。卡方检验适用于方差的检验。步骤如下：
    1）分均值已知与均值未知两种情况，求得卡方检验统计量
    2）根据备选假设以及α，求得所选假设对应的拒绝域
    3）根据1）中求得的卡方统计量与2）中求得的拒绝域，判断假设成立与否。
