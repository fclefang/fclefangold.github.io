二. 安装配置Geoserver

环境： Ubuntu16.04

1. 安装

  • 安装java
  • 安装tomcat
  • 下载geoserver-2.9.2-war.zip，解压该文件并将geoserver.war复制到/usr/local/tomcat/apache-tomcat-9.0.0.M11/webapps中去。
  • 重启tomcatsystemctl restart tomcat，浏览器输入http://服务器ip:8080/geoserver应该就能打开GeoServer了。默认用户名admin和geoserver密码登入就可以见到主界面。
  • 当然也可以不通过tomcat来启动geoserver服务，完全可以直接sh startup.sh，启动文件在bin目录下，所以tomcat是可选可不选的。
  • 更多的geserver配置可以注意web.xml和init文件，我们可以通过这两个文件修改端口以及登入账号密码等等。

2. 导入osm地图数据

  • openstreetmap中国数据下载地址
  • 有了地图数据，我们需要下载安装一个osm2pgsql工具，这个工具可以将地图数据导入到我们的数据库中，工具安装方法可以直接apt-get install osm2pgsql或者下载github源码，这里更多细节参考osm2pgsql的github地址

  • 下载地图数据的样式文件openstreetmap-carto，然后解压，最后复制到/home/postgresql_data中。

    • 通过osm2pgsql工具导入数据及其样式文件，其中pbf文件就是我们下载的地图数据文件。这里导入时间可能有点长，看电脑配置。
