JS深入浅出日记——One

JS深入浅出日记——One

  1. 数据类型
    object
    number
    string
    boolean
    null
    undefined
  2. 巧用+/-号实现类型转换
    for instance:var num;
    num-0 :将num转换成数字类型
    num+” :将num转换成字符串型
  3. JS引擎关于基本类型当对象使用时：
    var str=”string”;
    str.t=10;
    这里JS引擎将基本字符串类型str转换成对象str，并给t属性赋值，然 后销毁对象str。
    问题： 这种智能化将基本类型转换成对象有什么实际意义？
  4. 类型检测：
    typeof: 适合基本类型和函数对象，直接返回一个字符串，非常直观。
    instanceof: 适合自定义对象，也可以用来检测原生对象，在不同的if rame和window之间检测是无效的。因为生成的是不同的对象！
    Object.prototype.toString
    Constructor
    duck type

  5. 将数组转为字符串：
    `Array.prototype.join:
    function repeatString(n,str){

    return new Array(n+1).join(str);

    }

    repeatString(3,a);//"aaa"`

  6. 排序：
    Array.prototype.sort: var arr=[13,24,51,3]; arr.sort();//[13,24,3,51]，将数字看成字符串排序 arr;//[13,24,3,51]，并不是新建一个数组排序，直接在原数组排序。
    那么如何将数组内的数字排序呢？
    添加一个比较函数：
    arr.sort(function(a,b){ return a-b; });
